Merge conflict di git Salah kegagalan Git untuk meulai proses merge file dan dapat terjadi ketika dua branches terpisah melakukan edit pada line yang sama dalam suatu file, atau ketika sebuah file telah diapus pada suatu branch, namun diedit pada branch yang lain. Merge conflict umum terjadi ketika bekerja dengan tim. Kegagalan merge dapat terjadi di awal maupun di tengah-tengah proses merge. Merge conflict dapat diatasi menggunakan command line, tools seperti git checkout, git reset —mixed, atau git merge —abort.
